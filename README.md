# Hexagonal architecture 
Projeto simples para demonstrar organização de pacotes e uso da arquitetura hexagonal

## Instalação

```shell
# build dos artefatos
mvn clean install 

# build images
sh docker-build-images.sh

# inicializa o ambiente
docker-compose -f docker-compose.yml up -d

```
